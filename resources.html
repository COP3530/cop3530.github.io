Academic Policies

Academic Honesty

  • ✓ Collaborate on understanding concepts
  • ✓ Discuss approaches and algorithms
  • ✓ Use online references with proper citation
  • ✗ Copy code from classmates or online
  • ✗ Submit identical or substantially similar work
  • ✗ Use AI tools to write code without disclosure

Coding Style Guide

  • • Use meaningful variable and function names
  • • Consistent indentation (4 spaces or tabs)
  • • Comment complex algorithms and logic
  • • Include header comments with author info
  • • Avoid magic numbers, use named constants
  • • Handle edge cases and error conditions
